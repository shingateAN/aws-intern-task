# AWS Intern Hands-On Task

This project demonstrates a highly available web application architecture on AWS using:

- Custom VPC
- Public and Private Subnets
- NAT Gateway
- Application Load Balancer
- Auto Scaling Group
- EC2 Instances
- Target Group
- NGINX using User Data

## Validation Performed
- Verified load balancing using ALB DNS
- Verified Auto Scaling self-healing capability
- Verified private subnet deployment
